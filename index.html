Resumo de Tags HTML

As tags HTML são utilizadas para estruturar e formatar o conteúdo em uma página web. Aqui estão as principais tags e suas funções:

Tags de Cabeçalho

<h1> até <h6>: Definem cabeçalhos, sendo <h1> o mais importante e <h6> o menos importante.

Tags de Texto

<p>: Define um parágrafo.

<mark>: Destaca o texto com um fundo amarelo.

<small>: Exibe o texto em tamanho menor.

<i>: Exibe o texto em itálico.

<u>: Sublinha o texto.

<strong>: Exibe o texto em negrito.

Listas

<ol>: Define uma lista ordenada (numerada).

<ul>: Define uma lista não ordenada (com marcadores).

<li>: Define um item de lista.

Links e Separadores

<a>: Define um hyperlink.

<hr>: Insere uma linha horizontal para separar conteúdos.

Subscrito e Sobrescrito

<sub>: Define texto subscrito (abaixo da linha base).

<sup>: Define texto sobrescrito (acima da linha base).

Citações e Formatação

<blockquote>: Define uma citação em bloco.

<font>: Altera a fonte do texto (não recomendado para uso atual).

<del>: Exibe o texto com um traço, indicando que foi deletado.

Outras Tags

<abbr>: Define uma abreviação, fornecendo uma descrição completa em um tooltip.
