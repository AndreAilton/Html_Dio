# Resumo de Tags HTML

Este trabalho apresenta um resumo das principais tags HTML utilizadas para estruturar e formatar o conteúdo em uma página web.

## Tags de Cabeçalho

- `<h1>` até `<h6>`

## Tags de Texto

- `<p>`
- `<mark>`
- `<small>`
- `<i>`
- `<u>`
- `<strong>`

## Listas

- `<ol>`
- `<ul>`
- `<li>`

## Links e Separadores

- `<a>`
- `<hr>`

## Subscrito e Sobrescrito

- `<sub>`
- `<sup>`

## Citações e Formatação

- `<blockquote>`
- `<font>`
- `<del>`

## Outras Tags

- `<abbr>`

## Entidades HTML

- `&lt;` e `&gt;`
